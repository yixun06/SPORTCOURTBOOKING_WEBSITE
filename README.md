# SportCourtBooking

SportCourtBooking is an ASP.NET Web Forms application for managing sports court reservations. It demonstrates booking workflows, schedule tracking, and booking history features in a clean .NET Framework project structure.

## Features
- Create sports court bookings
- View existing bookings and status
- Track booking history
- Server-side booking flow with ASP.NET Web Forms code-behind

## Tech Stack
- ASP.NET Web Forms
- C#
- .NET Framework 4.7.2
- HTML/CSS

## Project Structure
- SportsBooking.sln: Solution file
- SportsBooking/: Main web application
- SportsBooking/Booking.aspx: Booking page
- SportsBooking/History.aspx: Booking history page
- SportsBooking/Default.aspx: Landing page

## Run Locally
1. Open SportsBooking.sln in Visual Studio.
2. Restore NuGet packages.
3. Build the solution.
4. Run with IIS Express (F5).
5. Open Booking and History pages to validate core flow.
